# Sprint 1 Production Playbook Skeleton

Status: Skeleton  
Pillar: Load Reduction  
Sprint: 1

---

## Operating Rule

Founder reviews and approves playbook use, but execution is delegated to production team roles.

## 1) Pre-Production Checklist

- Scope confirmed against proposal
- Technical requirements validated
- Crew assignments drafted
- Equipment pull list validated
- Venue constraints and access times confirmed

## 2) Crew Brief Template

- Event summary
- Team roles and chain of command
- Call times
- Safety notes
- Escalation protocol

## 3) Load-In Timeline

- Arrival window
- Staging sequence
- System build milestones
- Line-check and show-readiness checkpoints
- Client sign-off checkpoint

## 4) Show-Day Checklist

- System health check
- Show cues/run-of-show confirmation
- Communication channels live
- Onsite issue log active
- Client touchpoints scheduled

## 5) Strike Checklist

- Power-down sequence
- Gear pack-out accountability
- Venue handoff complete
- Damage/loss report logged

## 6) Post-Event Margin Review

- Revenue captured
- Direct costs captured
- Gross margin snapshot
- Variance drivers noted
- Improvement actions assigned

---

## Compliance Checklist

- [ ] All six sections completed before event
- [ ] Crew brief issued before load-in
- [ ] Margin review completed within 48 hours post-event
